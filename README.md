# cf-finder
FInd correct output for given input.
Learning purposes only.

Used
* React
* Babel
* Webpack


Setup dev environment
* fork and clone
* ``npm install``
* ``npm start`` #(starts dev server on 8080 port)
* Push after running ``npm run build``
* Make a PR
